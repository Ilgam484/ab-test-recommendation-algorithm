# 📊 Анализ A/B-теста: влияние алгоритма рекомендаций на CTR

## 📌 Описание проекта

В рамках проекта был проанализирован A/B-тест нового алгоритма рекомендаций постов в новостной ленте.

Цель эксперимента — определить, приводит ли новый алгоритм к увеличению CTR (Click-Through Rate).

---

## 🎯 Бизнес-задача

Проверить гипотезу:

> Новый алгоритм рекомендаций увеличивает CTR пользователей.

---

## 🧪 Данные

- Период эксперимента: 2026-01-30 — 2026-02-05  
- Группа 1 — контроль  
- Группа 2 — тест (новый алгоритм)  

Использованы данные пользовательских действий:
- просмотры постов  
- лайки постов  

---

## 📊 Метрика

**CTR (Click-Through Rate)**:

CTR = likes / views

---

## 🔬 Методы анализа

Для оценки различий между группами использованы:

- t-test  
- тест Манна-Уитни  
- Пуассоновский бутстреп  
- t-test на сглаженном CTR (α = 5)  
- бакетное преобразование + t-test  
- бакетное преобразование + Манна-Уитни  

---

## 📈 Анализ данных

- Построены распределения CTR по группам  
- Выявлены различия в форме распределений  
- Проверены предположения статистических тестов  
- Проведено сравнение результатов разных методов  

---

## 🔍 Результаты

- Разные методы показали неоднозначные результаты  
- Часть тестов выявляет статистически значимые различия, часть — нет  
- Распределения CTR имеют особенности, влияющие на чувствительность тестов  

---

## ⚠️ Интерпретация

- CTR имеет нестандартное распределение  
- чувствительность тестов зависит от метода  
- наличие выбросов и асимметрии влияет на результаты  

---

## 💡 Вывод

Нельзя однозначно утверждать, что новый алгоритм улучшает CTR.

Рекомендуется:

- провести дополнительный анализ  
- использовать более устойчивые метрики  
- проверить эффект на других сегментах пользователей
